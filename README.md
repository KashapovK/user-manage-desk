# User Management App  

Двухстраничное React-приложение для управления пользователями с адаптивным дизайном и современным стеком технологий. Проект предоставляет интерфейс для управления пользователями, а также для просмотра и редактирования данных пользователей.  

## Функциональность  

Главная страница отображает список пользователей, полученных с JSONPlaceholder API. На странице представлены две секции: активные пользователи и архив. Для каждой карточки пользователя доступны действия: редактирование, архивация и скрытие. Карточки из архива можно восстановить в активный список.  

Страница редактирования позволяет изменять данные пользователя: имя, никнейм, email, город, телефон и название компании. Реализована валидация всех полей формы. При успешном сохранении изменений появляется popup-уведомление, которое автоматически закрывается через 4 секунды или по клику пользователя.  

## Технологии  

Проект разработан на React с TypeScript. Для управления состоянием используется Zustand, работа с формами реализована через React Hook Form с валидацией Zod. API-запросы выполняются с помощью TanStack Query, роутинг организован через React Router v7. Стили написаны на SCSS с использованием адаптивных техник.  

В проекте реализована ленивая загрузка компонентов для оптимизации производительности. Шрифты предзагружаются. SVG иконки импортируются как React-компоненты через vite-plugin-svgr. Во время загрузки данных отображается кастомный лоадер-спиннер для улучшения пользовательского опыта.  

## Адаптивность  

Приложение полностью адаптировано для всех устройств. Для плавных переходов между размерами экранов применены fluid-функции вместо резких брейкпоинтов.  

## Деплой  

Приложение задеплоено на Vercel и доступно по ссылке [user-manage-desk.vercel.app](https://user-manage-desk.vercel.app) или в разделе about.  

## Архитектура  

Проект следует компонентному подходу с переиспользуемыми UI-элементами. Бизнес-логика вынесена в кастомные хуки, глобальное состояние управляется через стор-паттерн. Стили организованы через SCSS модули для изоляции стилей компонентов.  

## Установка и запуск  

Для запуска проекта необходимо клонировать репозиторий, установить зависимости через npm install и запустить development сервер командой npm run dev. Для production сборки используется команда npm run build.
